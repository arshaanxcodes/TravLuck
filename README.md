# Lucknow Scrollytelling Website

Pure `HTML + CSS + JavaScript` immersive travel story website for Lucknow.

## Stack

- HTML5
- Modern CSS (responsive + animated)
- Vanilla JavaScript
- GSAP (CDN) for extra scroll animation polish
- Leaflet (CDN) for interactive map

## Features

- Fullscreen cinematic hero
- Animated timeline history section
- Landmarks cards with modal details
- Scroll-driven food storytelling rail
- Culture section with typewriter poetry
- Interactive map with animated pins
- Final CTA section
- Dark mode toggle
- Background music toggle
- Scroll progress bar
